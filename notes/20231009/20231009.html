an ender, a midi controller, and a p5.js sketch walk into a bar
last edited: 2023.10.09
  • Set up basic connection between the Akai Midimix Control Surface and an ender using p5.fab. This is the same midi controller that bunnie is using for jubiris.
  • Reading in midi values in e.g. a p5 sketch is straightforward using WebMidi. Midi messages have a few dimensions that can be useful including message type, note, and velocity. Data messages are values 0-127; mapping to e.g. 0-220 to control the x-axis of a printer loses a bit of resolution.
  • Messages come individually. For the axis control shown above I'm consolidating & sending messages every ~100ms so that movement happens on multiple axis simultaneously, rather than many small movements on each axis.
